# ProfitView

> A financial dashboard for small-to-medium reseller businesses.

## Overview

ProfitView helps reseller businesses track their daily sales, 
inventory, and expenses in one place — with real-time profit 
calculation and visual performance charts.

Built as a portfolio project to demonstrate frontend development 
and business logic implementation.

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Bootstrap 5.3
- Chart.js 4.4

## Features (V1)

- [ ] Dashboard with daily sales summary and charts
- [ ] Sales tracking with inventory validation
- [ ] Inventory management
- [ ] Expense tracking
- [ ] Asset / payment method management
- [ ] Transaction history with date filter
- [ ] CSV export
